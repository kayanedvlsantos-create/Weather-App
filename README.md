# 🌤️ Weather App

Aplicação web de clima desenvolvida em JavaScript puro consumindo a API Open-Meteo.

## 🚀 Funcionalidades

- Busca por cidade e estado
- Clima atual em tempo real
- Previsão para 7 dias
- Ícones dinâmicos
- Histórico de buscas

## 🛠️ Tecnologias

- HTML
- CSS
- JavaScript
- Open-Meteo API
