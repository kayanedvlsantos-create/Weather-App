# 🌤️ Weather App

Aplicação web de clima desenvolvida em JavaScript puro que consome a API da Open-Meteo para exibir dados meteorológicos em tempo real e previsão para os próximos dias.

---

## 🚀 Funcionalidades

- 🔎 Busca por cidade e estado
- 📍 Uso da localização atual do usuário
- 🌡️ Clima atual em tempo real
- 📅 Previsão para 7 dias
- 🔍 Autocomplete de cidades
- ⏳ Indicador de carregamento (loading)
- 🕘 Histórico de buscas
- 🎯 Interface simples e intuitiva

---

## 🛠️ Tecnologias utilizadas

- HTML5  
- CSS3  
- JavaScript (ES Modules)  
- API Open-Meteo
